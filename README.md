# 🎨 Neon Glow Icon Pack

A modern Android icon pack featuring minimal, flat design with dark backgrounds and vibrant neon glow accents.

## ✨ Design Style

- **Shape:** Squircle (superellipse) with smooth rounded corners
- **Background:** Dark charcoal-black
- **Logo:** Clean white vector icons
- **Effect:** Neon glow halo + soft drop shadow
- **Size:** 1024 × 1024 px (PNG, transparent background)
